# 运算符

**[English](operators.md)** | **中文**

---

OLang 是**强静态类型**语言：对二元运算，左右两边必须是**同一种类型**；表达式里**不会做隐式类型提升**，需要别的类型时用 `T(…)`（**值**转换）或 `let n <T>x`（同址换类型视图）。

下表按**类型**列出编译器（`sema.c`）实际允许的运算符。

---

### `bool`

| 运算符 | 含义 | 结果类型 |
|--------|------|----------|
| `!a` | 逻辑非 | `bool` |
| `a && b`、`a \|\| b` | 逻辑与/或（两侧均为 `bool`） | `bool` |
| `a == b`、`a != b` | 相等判断 | `bool` |

`bool` 没有大小比较（`<`、`>` 等）。

---

### 整数（`i8` … `i64`，`u8` … `u64`）

两侧必须是**同一种**整数类型（例如两个 `i32`，不能在同一表达式里混 `i32` 与 `u32`）。

| 运算符 | 含义 | 结果类型 |
|--------|------|----------|
| `-a` | 取负 | 与操作数相同 |
| `a + b`、`a - b`、`a * b`、`a / b`、`a % b` | 算术 | 与操作数相同 |
| `a < b`、`a > b`、`a <= b`、`a >= b` | 有序比较 | `bool` |
| `a == b`、`a != b` | 相等判断 | `bool` |

**整数上没有** `&`、`|`、`^`、`<<`、`>>`（按位运算只作用于下面的 `b*` 类型）。若要做按位操作，请使用 `b8`…`b64`，或在弄清规则后用显式的 **`bN(…)`** / **`uN(…)`** 值转换。详见 [types_zh](types_zh.md)。

---

### 浮点（`f16`、`f32`、`f64`）

两侧必须是**同一种**浮点类型。

| 运算符 | 含义 | 结果类型 |
|--------|------|----------|
| `-a` | 取负 | 与操作数相同 |
| `a + b`、`a - b`、`a * b`、`a / b` | 算术 | 与操作数相同 |
| `a < b`、`a > b`、`a <= b`、`a >= b` | 有序比较 | `bool` |
| `a == b`、`a != b` | 相等判断 | `bool` |

**没有** `%`（取余仅适用于整数与 `b*`）。

---

### 位向量（`b8`、`b16`、`b32`、`b64`）

两侧必须是**同一种** `b*` 类型。该类型既支持按「字」做算术，也支持按位运算。

| 运算符 | 含义 | 结果类型 |
|--------|------|----------|
| `-a` | 取负 | 与操作数相同 |
| `~a` | 按位取反 | 与操作数相同 |
| `a + b` … `a % b` | 算术（与整数同类，含 `%`） | 与操作数相同 |
| `a & b`、`a \| b`、`a ^ b`、`a << b`、`a >> b` | 按位与/或/异或/移位 | 与操作数相同 |
| `a == b`、`a != b` | 相等判断 | `bool` |

**没有** 有序比较（`<`、`>`、`<=`、`>=`）。

---

### `ptr`

| 运算符 | 含义 | 结果类型 |
|--------|------|----------|
| `a == b`、`a != b` | 地址是否相同 | `bool` |

其它二元运算符不能用于 `ptr`。

---

### 聚合类型（结构体、数组）

结构体与数组在检查器中**不算**可与算术/比较混用的标量：不能在表达式里对整体做 `==` / `!=` 或算术。写入使用 **`store[…]`**（左值可为字段链、下标等路径）。若要比较，请比较字段，或取字段/下标得到标量。

---

### 优先级（各类型通用）

高 → 低：

```
() [] . addr[find] sizeof load <T> T()      // 最高（T() 值转换；<T> 新建引用并绑定元素类型）
! ~ -                               // 一元
* / %
+ -
<< >>
< > <= >=
== !=
&
^
|
&&
||                                  // 最低
```

---

### 示例

```olang
let i stack[32, (a + b) * c];
let cmp stack[8, (i > 0i32) && done];     // i32 比较；bool 逻辑

let x stack[32, 0xFF00b32];
let y stack[32, x & 0x00FFb32];             // 按位运算只在 b* 上
```
