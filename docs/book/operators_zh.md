# 运算符

**[English](operators.md)** | **中文**

---

### 算术

| 运算符 | 示例 | 说明 |
|--------|------|------|
| `+` | `a + b` | 加法 |
| `-` | `a - b` | 减法 |
| `*` | `a * b` | 乘法 |
| `/` | `a / b` | 除法（整数） |
| `%` | `a % b` | 取模 |
| `-x` | `-a` | 一元负号 |

### 比较

| 运算符 | 说明 |
|--------|------|
| `==` | 等于 |
| `!=` | 不等于 |
| `<` | 小于 |
| `>` | 大于 |
| `<=` | 小于等于 |
| `>=` | 大于等于 |

返回 `bool`。

### 逻辑

| 运算符 | 说明 |
|--------|------|
| `&&` | 逻辑与 |
| `||` | 逻辑或 |
| `!` | 逻辑非 |

```olang
if (a > 0i32 && b > 0i32) { }
if (!done) { }
```

### 位运算

| 运算符 | 示例 | 说明 |
|--------|------|------|
| `&` | `a & b` | 按位与 |
| `|` | `a | b` | 按位或 |
| `^` | `a ^ b` | 按位异或 |
| `<<` | `a << n` | 左移 |
| `>>` | `a >> n` | 右移 |

### 优先级

高 → 低：

```
() [] . addr cast load      // 最高
! -                         // 一元
* / %
+ -
<< >>
< > <= >=
== !=
&
^
|
&&
||                          // 最低
```

### 示例

```olang
let x: i32 = (a + b) * c;           // 括号明确优先级
let y: bool = (x > 0i32) && flag;   // 比较优先于逻辑
let z: i32 = a & 0xFFi32;           // 取低8位
```
