### 内置类型

| 类型 | 大小 | 说明 |
|------|------|------|
| `void` | 0 | 无值 / 无返回值 |
| `bool` | 1 字节 | 布尔 |
| `i8`, `i16`, `i32`, `i64` | 1 / 2 / 4 / 8 字节 | 有符号整数 |
| `u8`, `u16`, `u32`, `u64` | 1 / 2 / 4 / 8 字节 | 无符号整数 |
| `f16`, `f32`, `f64` | 2 / 4 / 8 字节 | 浮点（IEEE binary16/32/64） |
| `b8`, `b16`, `b32`, `b64` | 1 / 2 / 4 / 8 字节 | 定宽原始位向量 |
| `ptr` | 8 字节 | 裸指针 |

### 字面量后缀

**整数：** 可选后缀 `i8`、`i16`、`i32`、`i64`、`u8`、`u16`、`u32`、`u64`、`b8`、`b16`、`b32`、`b64`。无后缀的十进制字面量默认为 `i32`（会做范围检查）。

**浮点：** 十进制，可有小数部分和/或指数；可选后缀 `f16`、`f32`、`f64`。无后缀时字面量类型为 `f64`。

```olang
42           // i32
42i64        // i64
42u64        // u64
255b8        // b8
3.14f64      // f64
18446744073709551615u64  // UINT64_MAX
```

### 类型定义

#### 结构体

```olang
type Point = struct { x: i32, y: i32 };
type Rect = struct { tl: Point, br: Point };
```

- 字段顺序声明
- 支持嵌套
- 赋值使用值拷贝

#### 数组

```olang
type IntArray = array(i32, 10);
```

- 编译期确定大小
- 变量实际为指针
- 不检查越界

### 类型转换

#### 内建 `目标类型(表达式)`（值位置）

没有 `cast` 关键字。**值位置**的 **`T(表达式)`** 只在内建类型 **`iN`、`uN`、`bN`、`fN`** 之间转换：各族内部的变宽/截断（`i*`↔`i*`、`u*`↔`u*`、`b*`↔`b*`、`f*`↔`f*`），以及 **`uN`/`iN` 与 `fN` 互转**。**不包含**与 **`ptr`**、**`bool`** 的值转换（用比较、控制流或引用视图）。**同宽度**下在不同族之间换解释（例如 `i32` 与 `u32`、或与同宽的 `bN`）**不是**值转换，应使用 **`let n <T>x`**、链式 **`let`** 等。对**单个裸变量**的同宽换视图不能写 **`T(x)`**，应使用 **`let n <T>x`**（**`<T>`** 在**右侧**引用上）、**`let a let b x`** 或 **`struct`** 字段（**不在** **`stack[…]`** 里列多类型）。

```olang
i32(i64_value)   // 截断
i32(i8_value)    // 扩展
u32(200u8)       // 无符号变宽
f32(3i32)        // 整数到浮点（含与 i32 同总位宽的情形）
```

#### 同宽存储多视图：`addr` + `<T>`

对已有存储再绑一个名字时，写 **`let n <T>x`**：`n` 是**新**引用，**`<T>`** 在这条新引用上绑定元素类型；`addr[x]` 只说明存储从哪来。或在**右端引用**上 **`let a let b x`**，或用 **`struct`** 同一块布局上的多个字段。

```olang
let x stack[32, -1];
let u <u32>x;
```

### 类型分类

| 分类 | 类型 | 初始化要求 |
|------|------|-----------|
| 标量 | 除 `void` 外的内置类型 | `let` 时必须初始化 |
| 聚合 | struct、array | 可延后初始化 |
