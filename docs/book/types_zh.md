### 内置类型

| 类型 | 大小 | 说明 |
|------|------|------|
| `void` | 0 | 无值 / 无返回值 |
| `bool` | 1 字节 | 布尔 |
| `i8`, `i16`, `i32`, `i64` | 1 / 2 / 4 / 8 字节 | 有符号整数 |
| `u8`, `u16`, `u32`, `u64` | 1 / 2 / 4 / 8 字节 | 无符号整数 |
| `f16`, `f32`, `f64` | 2 / 4 / 8 字节 | 浮点（IEEE binary16/32/64） |
| `b8`, `b16`, `b32`, `b64` | 1 / 2 / 4 / 8 字节 | 定宽原始位向量 |
| `ptr` | 8 字节 | 裸指针 |

### 字面量后缀

**整数：** 可选后缀 `i8`、`i16`、`i32`、`i64`、`u8`、`u16`、`u32`、`u64`、`b8`、`b16`、`b32`、`b64`。无后缀的十进制字面量默认为 `i32`（会做范围检查）。

**浮点：** 十进制，可有小数部分和/或指数；可选后缀 `f16`、`f32`、`f64`。无后缀时字面量类型为 `f64`。

```olang
42           // i32
42i64        // i64
42u64        // u64
255b8        // b8
3.14f64      // f64
18446744073709551615u64  // UINT64_MAX
```

### 类型定义

#### 结构体

```olang
type Point = struct { x: i32, y: i32 };
type Rect = struct { tl: Point, br: Point };
```

- 字段顺序声明
- 支持嵌套
- 赋值使用值拷贝

#### 数组

```olang
type IntArray = array<i32, 10>;
```

- 编译期确定大小
- 变量实际为指针
- 不检查越界

### 类型转换

#### `cast<目标类型>(表达式)`

在**不同类型**之间做编译器允许的显式转换：同一整数族内的变宽/截断（`i*`↔`i*`、`u*`↔`u*`）、`b*`↔`b*`、`f*`↔`f*`、`bool`↔整数、浮点↔整数等。对**计算出来的值**仍可用 `cast` 做同宽标量转换（例如 `cast<u32>(x + 1i32)`）。对**单个变量**的同宽「换标签」不能写 `cast<…>(x)`，应使用 **`let n<T> <T>addr x`**，或在一串 `let` 里用多个 `名字<类型>` 共享一块分配（每多一个名字就多写一个 `let`）。

```olang
cast<i32>(i64_value)   // 截断
cast<i32>(i8_value)    // 扩展
cast<u32>(200u8)       // 无符号变宽
cast<u32>(x + 0i32)    // 同宽值转换（表达式，不能是裸变量名）
```

#### 同宽存储多视图：`addr` + `<T>`

对已有存储再绑一个名字时，写 **`let n<T> <T>addr x`**（`addr` 取址，外层 `<T>` 指定视图类型）；或在一串 `let` 里声明多个 `名字<类型>` 共享一块分配（每多一个名字就多写一个 `let`）。

```olang
let x<i32> @stack<32>(-1);
let u<u32> <u32>addr x;
```

### 类型分类

| 分类 | 类型 | 初始化要求 |
|------|------|-----------|
| 标量 | 除 `void` 外的内置类型 | `let` 时必须初始化 |
| 聚合 | struct、array | 可延后初始化 |
