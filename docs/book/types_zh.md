### 内置类型

| 类型 | 大小 | 范围 | 说明 |
|------|------|------|------|
| `void` | - | - | 无返回值 |
| `bool` | 1字节 | true/false | 布尔 |
| `u8` | 1字节 | 0..255 | 无符号字节 |
| `i32` | 4字节 | -2^31..2^31-1 | 有符号32位 |
| `u32` | 4字节 | 0..2^32-1 | 无符号32位 |
| `i64` | 8字节 | -2^63..2^63-1 | 有符号64位 |
| `u64` | 8字节 | 0..2^64-1 | 无符号64位 |
| `ptr` | 8字节 | 地址 | 裸指针 |

### 字面量后缀

| 后缀 | 类型 | 范围 |
|------|------|------|
| (无) | `i32` | -2^31..2^31-1 |
| `i32` | `i32` | -2^31..2^31-1 |
| `u32` | `u32` | 0..2^32-1 |
| `i64` | `i64` | -2^63..2^63-1 |
| `u64` | `u64` | 0..2^64-1 |
| `u8` | `u8` | 0..255 |

```olang
42           // i32
42i64        // i64
42u64        // u64
18446744073709551615u64  // UINT64_MAX
```

### 类型定义

#### 结构体

```olang
type Point = struct { x: i32, y: i32 };
type Rect = struct { tl: Point, br: Point };
```

- 字段顺序声明
- 支持嵌套
- 赋值使用值拷贝

#### 数组

```olang
type IntArray = array<i32, 10>;
```

- 编译期确定大小
- 变量实际为指针
- 不检查越界

### 类型转换

```olang
cast<DestType>(source)
```

- 显式转换
- 位模式保留
- 可整数↔指针互转

```olang
cast<i32>(i64_value)     // 截断
cast<u32>(-1i32)         // 位模式 reinterpret
cast<ptr>(0x1000u64)     // 整数转指针
```

### 类型分类

| 分类 | 类型 | 初始化要求 |
|------|------|-----------|
| 标量 | i32, i64, u32, u64, u8, bool, ptr | `let` 时必须初始化 |
| 聚合 | struct, array | 可延后初始化 |
